Install

Get started.

1
Download the extension from the Chrome Web Store or clone the GitHub repository.
2
If loading manually, open chrome://extensions and enable Developer mode.
3
Click Load unpacked and select the extension folder.
4
Browse normally. Names from the list will be highlighted automatically on every page.
