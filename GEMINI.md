# Gemini Agent Instructions

## Memory Maintenance
1. **Initialize a Memory File**: For every new project, start by creating a new memory markdown file (e.g., `PROJECT_MEMORY.md` or `MEMORY.md`) in the project's root directory.
2. **Log Failed Attempts**: Each time an initial attempt at a task, fix, or implementation does not work, you must document the failure in the memory file.
3. **Document the Solution**: Alongside the noted failure, record the actual solution or workaround that eventually succeeded. This ensures the agent learns from mistakes and builds a reliable knowledge base for the project.
